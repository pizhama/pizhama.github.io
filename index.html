Инструкция по использованию:

  1. Зарегистрируйтесь на GREEN-API и создайте инстанс в личном кабинете
  2. Отсканируйте QR-код для подключения вашего номера WhatsApp
  3. Введите ваши учетные данные: ID инстанса и API Токен
  4. Введите номер телефона получателя в формате 7XXXXXXXXXX
  5. Используйте кнопки для вызова методов API и просматривайте результаты в поле ответа
